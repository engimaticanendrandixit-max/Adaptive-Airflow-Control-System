# Smart Fan V1.0 🚀

An ESP32-based Smart Fan prototype developed as an Electrical Engineering project. This project demonstrates fan speed control, RPM monitoring, power consumption measurement, and environmental monitoring using ESP32.

---

## 📌 Overview

Smart Fan V1.0 is a low-cost prototype that combines sensors and an ESP32 microcontroller to monitor and control a PWM fan in real time.

The system measures:

- Fan speed (RPM)
- Temperature
- Humidity
- Current consumption
- Power consumption

All values are displayed on an OLED screen.

---

## ✨ Features

✅ ESP32-based fan controller

✅ PWM speed control

✅ Humidity sensing
